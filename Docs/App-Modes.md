# Режимы работы
В приложении реализованы 2 типа режимов работы:
- **AppMode** - начальный запуск.
- **Storage: Mode** - хранение данных.

Данные ключи находятся в блоках **Settings** и **Storage**.

## AppMode
**Описание**: Ключ, влияющий на то, как будет открываться приложение и какой пользователь используется. Находится в блоке **Settings**.

**Режимы работы**:
- **Auth** - при запуске открывается окно авторизации.
- **NoAuth** - сразу открывается главное окно, берётся базовый пользователь.

Настройка в **appsetting.json**:
``` json
    "Settings": {
        "AppMode" : "Auth"
    },
```

## Storage: Mode
**Описание**: Ключ, влияющий на то, как именно будут сохраняться заметки. Находится в блоке **Storage**.

**Режимы работы**:
- **DatabaseAndFile** - сохранение происходит как в базу данных, так и в файлы.
- **DatabaseOnly** - сохранение происходит только в базу данных.
- **FileOnly** - сохранение происходит только в файлы.

Настройка в **appsetting.json**:
``` json
    "Storage": {
        "Mode": "DatabaseAndFile",
        "NotesPath": "Notes"
    }
```
